A lot of current web apps and mobile apps utilize a back-end server for communication and updating data. In this case, we can utilize a web proxy to capture HTTP and HTTPS traffic for analysis and ultimately act as sort of a Mitm. 

-------
### Burpsuite

 - Most common web proxy for penetration testing. Excellent user interface for its various features and even provides a built-in Chromium browser to test web application. 
- There is a free and paid version of Burpsuite. 

--------
### OWASP Zed Attack Proxy (ZAP)
- Another common web proxy tool for web penetration testing. 
- Biggest pro is that it is free compared to that of Burp Pro which is a paid program.
